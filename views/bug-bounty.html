🐛 Bug Bounty Program

Program pembelajaran keamanan siber untuk clone Facebook tempatan

🎯 Scope (Ruang Lingkup)

Aplikasi ini adalah clone Facebook tempatan untuk tujuan pembelajaran bug bounty. Semua risiko di bawah ini adalah sengaja dikekalkan untuk latihan:

⚠️ PERINGATAN: Ini adalah environment LATIHAN. Jangan gunakan teknik yang diperoleh di sini pada sistem sebenar tanpa izin.

Vulnerabilities yang sengaja dikekalkan:

  • IDOR - Postingan dan data pengguna boleh diakses tanpa authorization yang betul
  • XSS - Tiada sanitization pada input pengguna (postingan, komen, nama)
  • CSRF - Tiada CSRF token pada operasi yang mengubah state
  • Session Fixation - Session ID tidak diubah selepas login
  • Information Disclosure - Metadata API mendedahkan struktur internal
  • Broken Access Control - Tiada verifikasi pemilik pada operasi edit/delete
  • Rate Limiting - Tiada had pada percubaan login atau API calls
  • Insecure Direct Object Reference - Boleh akses /api/posts/:id dan /api/users/:id secara berurutan
📋 Panduan Pengujian

1. Daftar / Login

Buat akaun dengan username dan password mana sahaja:

POST /api/auth/register
{"username":"hacker","name":"Bug Hunter","password":"test123"}

2. Uji IDOR

Cuba akses postingan pengguna lain dengan menukar ID:

GET /api/posts/<OTHER_USER_POST_ID>
PUT /api/posts/<OTHER_USER_POST_ID>
DELETE /api/posts/<OTHER_USER_POST_ID>

3. Uji XSS

Cuba input payload XSS dalam:

  • Nama profil: <script>alert(1)</script>
  • Konten postingan: <img src=x onerror=alert(1)>
  • Komentar: <svg onload=alert(1)>

4. Uji CSRF

Cuba buat request tanpa token CSRF:

POST /api/posts (tanpa CSRF token)
POST /api/posts/:id/like (tanpa CSRF token)

5. Uji Session

Cuba akses protected routes tanpa session:

GET /api/auth/me (tanpa cookie session)
POST /api/posts (tanpa cookie session)
